Create the tutorial starter:

aflat make tutorial

Use --lib before make for a library template. The generated folder ships with aflat.cfg, src/main.af, and src/test/test.af.

Adding dependencies

List modules under [dependencies] in aflat.cfg. Run aflat install <git_repo> to clone remote repositories and update the config automatically.

Hello world breakdown

.needs <std>
import * from "io" under io;

fn main() -> int {
    io.print("Hello, world!\n");
    return 0;
};
  • .needs brings in the std library.
  • import exposes io helpers under the io. prefix.
  • fn main is the entry point; io.print writes to stdout.
