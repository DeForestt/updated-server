Installation

Remember: If aflat is already installed, pull the latest changes first. Refer to the README instructions to download and build the toolchain.

Update in place:

aflat update

Build the compiler from source with CMake:

mkdir build && cd build
cmake .. && make

Format C++ changes with clang-format and run ./rebuild-libs.sh after editing libraries/std/src. Use ./bin/aflat run to ensure everything works.
