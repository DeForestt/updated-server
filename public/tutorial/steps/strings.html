Working with strings

.needs <std>
import string from "String";
import { readString, print } from "String" under str;

fn main() -> int {
    let name = str.readString("What is your name? ");
    str.print(`Hello, {name}.\nFrom the AFlat Team.\n`);
    return 0;
};
  • Import string for the type and targeted helpers under str.
  • let infers types; prompts pass directly into readString.
  • Backticks enable interpolation: {name} expands inline.
