Ownership model

AFlat tracks ownership for heap objects:

  • Only owned values (from new or &&) can be moved or returned.
  • Use $ to transfer ownership.
  • && parameters consume their arguments.
  • Returning fields transfers ownership; avoid double-use after moves.
Expression                   Owned? Requires $?
---------------------------------------------
new Foo()                    yes    no
let f = new Foo()            yes    yes
someFunc() returning T!      yes    no
field from owned object      yes    yes
stack or literal value       no     n/a
function param (&&)          yes    yes or no (if rvalue)
